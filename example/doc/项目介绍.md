# task_armPickPlace 项目介绍

本项目是一个基于 Isaac Sim 5.1 的机械臂抓取与分类任务演示程序，实现了 Franka 机械臂在仿真环境中自动抓取彩色方块并按颜色分类堆叠的完整流程。

## 核心功能

程序通过模块化设计实现了完整的机器人操作任务：在仿真场景中随机生成多个彩色立方体（红、绿、蓝三种颜色），Franka 机械臂依次抓取这些方块，并根据方块颜色将其放置到对应的目标位置，同时支持堆叠功能，能够将同色方块垂直堆叠。

## 技术架构

项目采用 Isaac Sim 的标准任务框架，包含三个核心模块：

1. **主程序模块** (`main_task_armpickplace.py`)：负责启动 Isaac Sim 仿真应用，创建 World 和 Scene，初始化任务环境与控制器，并运行主仿真循环。程序实现了完整的观测-控制-执行流程，支持仿真状态管理和周期性指标计算。

2. **任务环境类** (`class_taskEnv.py`)：继承自 `BaseTask`，负责场景初始化、观测数据采集和任务指标计算。该类实现了随机方块生成（避免重叠）、Franka 机械臂加载、多维度观测获取（关节状态、末端位姿、方块位置与颜色）以及任务完成条件判断等功能。

3. **控制器类** (`class_controller.py`)：基于 `BaseController` 和 `PickPlaceController`，实现了高层抓取放置策略。控制器管理方块抓取顺序，根据方块颜色索引确定目标放置位置，动态计算堆叠高度，并在完成每个方块操作后自动切换到下一个目标。

## 技术特点

- **模块化设计**：采用任务-控制器分离架构，便于扩展和修改
- **鲁棒性**：包含异常处理机制，支持机器人加载失败时的容错运行
- **可配置性**：支持自定义方块数量、尺寸、目标位置等参数
- **完整观测**：提供机器人关节状态、末端执行器位姿、方块位姿与颜色等多维度观测数据

## 运行方式

项目提供了便捷的运行脚本 `autorun.sh`，自动配置 Isaac Sim 环境并启动程序。该脚本使用 Isaac Sim 的 `python.sh` 确保正确的 Python 环境和依赖路径，用户只需执行 `./autorun.sh` 即可启动仿真。

本项目可作为学习 Isaac Sim 机器人仿真、任务设计、控制器开发的优秀示例，也可作为强化学习训练的基础环境进行扩展。
